# MC Scheduler

## How to Run
Assuming dependencies are met:
- MC-Schedule-Builder/client
  1. npm install
  2. npm run serve
- MC-Schedule-Builder/server
  1. source env/bin/activate
  2. python app.py
- Go to http://localhost:8080/
